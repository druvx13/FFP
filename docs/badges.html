HTML Usage

For HTML websites, use this snippet:

<a href="https://github.com/druvx13/FFP" target="_blank" rel="noopener noreferrer"> <img src="https://img.shields.io/badge/license-FFP%20v2.0-blue.svg" alt="FFP License"> </a>

Badge Best Practices

  • Always link badges back to the FFP license page or GitHub repository
  • Include descriptive alt text for accessibility
  • Place the badge in a visible location (README top section, footer, etc.)
  • Use a consistent badge style across your project
  • Add rel="noopener noreferrer" when the link opens in a new tab
